What it does

☁️

AWS, locally

LocalStack emulates AWS services on your machine, so you can develop and test against S3, SQS, Lambda and friends without touching a real account.

🐳

One command up

A pre-wired docker-compose.yml and Dockerfile — just docker-compose up --build and the stack is running.

📜

Auto-provisioning

On boot the container runs scripts/init.sh, which chains your child scripts in order to seed buckets, queues and whatever else you need.

💸

Zero cloud cost

Everything runs in a throwaway container. Tear it down, rebuild, and start clean — no orphaned cloud resources, no surprise charges.

Quick start

Install Docker (and the Compose plugin on Linux) and start the daemon.

Pull the image once with docker pull localstack/localstack.

Bring it up from the directory holding the compose file. The container starts and scripts/init.sh runs every child script automatically. 

# clone & enter
git clone \
  https://github.com/karantrehan3/docker-localstack
cd docker-localstack

# pull LocalStack
docker pull localstack/localstack

# spin it all up
docker-compose up --build
